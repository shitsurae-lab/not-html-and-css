# 「ゼロ」から web ページを組み立てるには

## コンテナの分割

- 横幅が変わることろで分割する
- 背景の塗りつぶし方法が変わるところで分割
- コンテナは原則、縦から積み重なる。(横方向や重なるということは原則無い)

## モジュールの作成

コンテナを情報の内容やレイアウト似合わせて分割する

##　無視すべき点
背景画像がある場合でも CSS で表現できる箇所(あしらいなど)の場合は分割箇所としては無視をする

## コンテナーのパターン化

```HTML
<div class="クラス名">
  <div class="contaiern">

  </div>
</div>
<!-- End クラス名-->
```

| タグ                |     |
| ------------------- | --- |

| <footer></footer>   |     |
| <main></main>       |     |
| <article></article> |     |
| <section></section> |     |
| <nav></nav>         |     |
| <div></div>         |     |
